# 目录结构
一级目录分为两类。
1. 知识点目录(如Andorid\Java\Html\CSS\.Net)。
2. assets目录(存放图片等)

## 1、Android目录

### 0.Android知识点大全
存放汇总知识，目前建设中...

### 1.Android基础进阶
存放基础原理层面的知识，比如View的事件机制、分发机制，RecyclerView的绘制原理与工作机制等等。

### 2.Android实践应用
存放实际开发过程中总结的内容。比如AS使用中的问题，RecyclerView怎么用，等等。

### 3.开源库
1. 开源库汇总。
2. 各种开源库的原理和底层机制的分析。如:EventBus的源码分析。
